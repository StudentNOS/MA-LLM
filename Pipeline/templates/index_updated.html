Initial PMIDs (.txt) Format

Eine einfache Textdatei (.txt) mit einer Liste von PubMed PMIDs. Jede PMID steht in einer eigenen Zeile. Leerzeilen und Kommentare werden ignoriert.

Beispiel:

Goldstandard PMIDs (.txt) Format

Eine einfache Textdatei (.txt) mit einer Liste der bekannten relevanten PubMed PMIDs. Jede PMID steht in einer eigenen Zeile. Leerzeilen und Kommentare werden ignoriert.

Beispiel:

12345678
34567890
×

Prompts (.xlsx) Format

Eine Excel-Datei (.xlsx) mit den Prompts für das Screening. Erforderliche Spalten: TitlePrompt, AbstractPrompt, screen_titles, screen_abstracts.

  • TitlePrompt: Prompt für das Screening basierend auf Titeln.
  • AbstractPrompt: Prompt für das Screening basierend auf Abstracts.
  • screen_titles: 1 wenn Titel gescreent werden sollen, sonst 0.
  • screen_abstracts: 1 wenn Abstracts gescreent werden sollen, sonst 0.

Beispiel:

TitlePromptAbstractPromptscreen_titlesscreen_abstracts
Identify articles about cancer therapy.Determine if this abstract discusses cancer treatment.11
Find papers on diabetes research.Check if the abstract covers diabetes studies.10
